🚀 Mars One Project — Interactive Space Experience

An immersive, scroll-driven web experience that simulates humanity’s journey from Earth to Mars. Built with modern frontend technologies to deliver a cinematic storytelling experience through motion, visuals, and design.

🌌 Overview

Mars One Project is a single-page interactive website that takes users through a visually engaging journey to Mars. Each section represents a stage of space travel—launch, deep space, and landing—enhanced with smooth animations and a minimalistic black-and-white space theme.

The project focuses on frontend storytelling, combining animation, layout, and user interaction to create a seamless and engaging user experience.

✨ Features
🌠 Smooth scroll-based storytelling experience
🚀 Visual journey from Earth to Mars
🌌 Clean black & white space-themed design
🎥 Parallax and animation effects using motion libraries
📱 Fully responsive across devices
🔗 External reference link integration (SpaceX)
⚡ Optimized performance and fast loading

🛠️ Tech Stack
React.js – Component-based architecture
Framer Motion – Animations and transitions
Tailwind CSS – Styling and layout design
Vite – Fast build and development environment

🚀 Getting Started
1. Clone the Repository
  git clone https://github.com/mohithchukkala/mars-one-project.git
  cd mars-one-project
2. Install Dependencies
  npm install
3. Run the Development Server
  npm run dev
4. Build for Production
  npm run build
